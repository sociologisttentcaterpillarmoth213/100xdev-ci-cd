## 📈 Improving your delivery speed

This tool works by creating a pipeline for your code. A pipeline is a series of automatic events. When you save a file, the application triggers a sequence. It builds the project. It runs automated tests. It deploys the result to your destination. 

You save time by skipping the manual build process. You reduce human error because the computer performs every check exactly the same way. You gain peace of mind knowing the software arrives at the destination in a stable state.
